# Sheryl Liu — Product Portfolio

Personal product management portfolio. Static site, no build step.

## Structure
- `index.html` — single-page portfolio (hero, experience, about, work, side projects, case studies, contact)
- `docs/portfolio-plan.md` — content plan and build roadmap

## Deploy (Vercel)
1. Import this repo at vercel.com/new
2. Framework preset: **Other** — no build command, output directory `./`
3. Every push to `main` auto-deploys

## Local preview
Open `index.html` in a browser, or `npx serve .`
